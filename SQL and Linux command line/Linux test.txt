Напишите команду для терминала

Ты работаешь в Linux и не знаешь, в какой директории сейчас находишься.
Напишите ОДНУ команду, которая создаст пустой текстовый файл /tmp/report.log

mkdir -p tmp && > tmp/report.log
Используя " -p ", будет создана папка и файл, а если папка tmp уже существует, то будет создан только текстовый файл. 
" > " вместо touch, потому что  такой же текстовый файл уже может существовать и в нем могут быть данные, поэтому лучше его переписать и получить пустой файл.
